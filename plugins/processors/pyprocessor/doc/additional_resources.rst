********************
Additional Resources
********************

This section contains a list of additional resources for |RSPYTHON|.
The following resources are not detailed in this documentation because
they contain their own README file.

Examples
^^^^^^^^

Examples are located under ``<RTI Gateway main folder>/examples/pyprocessor``.
If you have provided CMake with an installation directory, they will be under
``<CMake Installation directory>/examples/pyprocessor``.


Tests
^^^^^

Tests are located under ``<RTI Gateway main folder>/plugins/processors/pyprocessor/test``
If you have provided CMake with an installation directory, they will be under
``<CMake Installation directory>/test/pyprocessor``.


Resources
^^^^^^^^^

Resources are located under ``<RTI Gateway main folder>/resource/pyprocessor``
If you have provided CMake with an installation directory, they will be under
``<CMake Installation directory>/resource/pyprocessor``.
Resources may include a different set of useful things like the Python abstract
class ``proc.Processor``.
