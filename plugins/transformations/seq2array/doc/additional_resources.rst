********************
Additional Resources
********************

This section contains a list of additional resources for |RS_SEQ2ARRAY_TSFM|.
The following resources are not detailed in this documentation because
they contain their own README file.


Examples
^^^^^^^^

Examples are located under ``<RTI Connext Gateway main folder>/examples/seq2array``.
If you have provided CMake with an installation directory, they will be under
``<CMake Installation directory>/examples/tsfm_seq2array``.


Tests
^^^^^

Tests are located under ``<RTI Connext Gateway main folder>/plugins/transformations/seq2array/test``
If you have provided CMake with an installation directory, they will be under
``<CMake Installation directory>/test/tsfm_seq2array``.
